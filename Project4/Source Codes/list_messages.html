Persistent Messages:

Project Demonstration:

This simple web application demonstrates data persistence using Docker volumes. Any messages added here are stored in an SQLite database within the Docker container. The persistent-sqlite-data Docker volume ensures that this data survives even if the container is stopped and restarted.

To add a new message, navigate to /add/YOUR_MESSAGE in the URL (e.g., /add/Testing Persistence).
